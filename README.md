# DSCI-532_2025_15_WorldEducation

This project is a dashboard to explore and understand world education dataset from the [kaggle](https://www.kaggle.com/datasets/nelgiriyewithana/world-educational-data/data). The dashboard is designed specifically for a policy makers user who wants to 1) explore the data to design policies that improve educational outcomes 2) use the dashboard to support her decisions.
